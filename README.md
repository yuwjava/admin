# Dujiao-Next Admin

Dujiao-Next Admin is the management console for operating products, orders, users, payment channels, and system settings.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS

## What This App Does

- Dashboard and operational analytics
- Product/category/content management
- Order and fulfillment management
- Payment channel configuration
- Admin role and permission management

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run build:fullstack
```
